# Ynglarite: Rescindment
"Ynglarite: Rescindment" is the working name of the full concept that this demo is from. Please don't use anything from this (the code is likely awful and the artwork is mine and also bad).
## Shrouded Temple
This is a currently unfinished prototype being developed at a project.
This is currently only on here to make demonstrating the current progress easier for myself.
### Things that currently work
- Background Rendering
- Object Rendering
- UI Rendering
- Object Translations
- Object Animation
- Boundings
- Limited Camera Controls (with player controls tied in)
### Things that aren't implemented yet
- Hash table for instantiation
- Clickable Buttons
- Custom Cursor
- Loading Boundaries
- Combat
- Turn-based mode
- Decision-making algorithms
- Other rooms
- Actions
